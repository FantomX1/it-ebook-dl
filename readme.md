# it-book-dl

a python tool to download books from it-ebooks.info!!!!

just:

```bash
$ python setup.py install
```

then to search :

```bash
$ find-books TERM

1. TITLE
2. TITLE
3. TITLE
4. TITLE
```


and if you see one you like,

```bash 
$ find-books TERM NUM
```

will download the item number from the list of searchs for the term
